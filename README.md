# Mandy 3.0:star:  `内测ing`
### 特色
- 强大后台主题设置
- 一体化插件
- 手机域名跳转
- 兼容 https
- 广告、幻灯片后台设置
### 环境依赖
###### PHP 5.2+ (7.1 没试过)
###### WordPress 4.0+ （兼容最新4.8版本）

### 巨人肩膀上

### 目录结构描述
```
├── mandy_main/
│   ├── 404.php    // 页面错误页
│   ├── archive.php     // 文章归档页
│   ├── comments.php     // 评论
│   ├── footer.php     // 底部区域代码
│   ├── functions.php     // 函数
│   ├── header.php     // 头部区域代码
│   ├── index.php     // 首页
│   ├── list.php     // 文章列表
│   ├── main.js     // 主题JS代码
│   ├── page.php     // 页面
│   ├── search.php     // 搜索页
│   ├── single.php     // 普通文章页
│   └── style.css     // 主题样式表
├── class.php     // 主题后台设置类
├── mandy_ini.php     // 插件声明文件
├── Mobile_Detect.php     // 手机判断文件
├── option.php     // 后台设置配置文件
├── README.md     // 帮助文档
```
